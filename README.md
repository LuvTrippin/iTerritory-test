# test-task-1

Тестовое задание для [iТерритория](https://it.gov74.ru/): одностраничное приложение (SPA) с демонстрацией базовых требований к вёрстке, семантике, навигации и работе с внешним API.

## Описание проекта

Проект - SPA без серверной части. На главной странице описаны требования тестового задания и реализованы ключевые пункты: изображение с fallback, галерея с загрузкой фото через API, якорные ссылки и контактные ссылки в подвале. Вторая страница (`/about`) добавлена для выполнения требования о минимум двух страницах в приложении.

## Стек

- React 19
- TypeScript
- Vite
- React Router
- Axios
- react-image-gallery
- CSS Modules

## Функциональность

### Навигация и страницы

- **Главная** (`/`) — описание задания, якорные ссылки, блоки с изображениями и галереей.
- **About** (`/about`) — вторая страница приложения.
- **Layout** — общий каркас с `<header>`, `<nav>`, `<main>`, `<footer>`.
- Якорные ссылки на главной: `#gallery`, `#img-with-fallback`.

### Изображение с fallback

Компонент `ImageWithFallback` подменяет основное изображение на локальный placeholder из `src/assets/placeholder.jpg`, если:

- не передан `src`;
- произошла ошибка загрузки (`onError`).

### Галерея

Компонент `Gallery`:

- загружает 30 фотографий через Axios из [Picsum Photos API](https://picsum.photos/);
- отображает слайды с превью и подписью автора;
- поддерживает полноэкранный режим по кнопке и по клику на большое изображение.

### Контакты и внешние ссылки

В подвале сайта:

- ссылка на телефон (`tel:`);
- ссылка на email (`mailto:`);
- ссылка на репозиторий проекта на GitHub.

### Семантика и вёрстка

- Используются семантические теги: `header`, `nav`, `main`, `footer`, `section`, `article`, `h1`–`h3`, `ul`.
- Стили вынесены в CSS Modules, inline-стили не используются.

# test-task-2

Тестовое задание для [iТерритория](https://it.gov74.ru/): SPA React + TypeScript, демонстрирующее работу с API, авторизацию и работу с сложными состояниями.

## Описание проекта

Проект - SPA без серверной части. Главная страница содержит список постов с возможностью создания, редактирования и удаления. Страница `/login` с формой авторизации. Все формы валидируются с помощью `yup` 

## Стек

- React 19
- TypeScript
- Vite
- React Router
- Axios
- Tanstack Query
- Yup
- Zustand

## Функциональность

### Навигация и страницы

- **Главная** (`/`) — Шапка с отображением почты текущего пользователя и возможностью разлогиниться, список постов с реализованными на них CRUD операциями.
- **Login** (`/login`) — Страница авторизации с Yup валидацией.

### Работа с постами
Реализованы следующие функции:
- Создать новый пост (Открывается модальное окно с формой)
- Редактировать существующий пост (Открывается модальное окно с формой, текущие значения полей подставляются автоматически)
- Удалить пост

## Поднятие проекта

### Требования
